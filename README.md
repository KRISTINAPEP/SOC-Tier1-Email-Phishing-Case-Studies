# SOC Tier 1 Security Monitoring and Email Investigation Case Studies

This repository contains a collection of SOC Tier 1 case studies performed in a personal lab environment for educational and portfolio development purposes. Cases demonstrate realistic Tier 1 responsibilities across security monitoring, alert triage, email investigation, and incident documentation.

The goal of this repository is to showcase practical analytical skills including log review, phishing detection, authentication analysis, link inspection, escalation decision making, and clear professional reporting.

## Scope of Work

This repository includes case studies involving:

- Email phishing analysis
- Security monitoring and alert triage
- Header and authentication evaluation
- Suspicious activity investigation
- Link and redirect analysis
- SOC documentation and escalation workflow
- Sanitized evidence handling

Each case is fully documented in a dedicated Markdown file containing:

- Case summary and scope
- Detailed investigation and findings
- Sanitized indicators and evidence
- SOC actions and final determination

## Purpose

These case studies are designed to demonstrate practical SOC Tier 1 analytical skills in a structured and professional format. They reflect realistic investigation workflows and emphasize clarity, accuracy, and responsible handling of evidence.

## Disclaimer

All investigations were performed in a personal non enterprise environment for educational and portfolio building purposes. Any domains, email addresses, or indicators referenced are limited to publicly observable attacker infrastructure. No confidential, proprietary, or organizational data is included.
